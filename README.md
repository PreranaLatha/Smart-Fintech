# Smart-Fintech

This repository contains the complete source code of a Fintech Microservices Platform.

## Services
- Auth Service (OAuth2)
- Card Service
- Transaction Service
- Payment Gateway Service
- Notification Service

## Tech Stack
- Java, Spring Boot
- Microservices Architecture
- Kafka
- MySQL
- Docker & Kubernetes
- Prometheus + Grafana (Monitoring)
- AWS Deployment
